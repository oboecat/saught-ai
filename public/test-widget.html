Saught AI Widget Test Page

This page tests the Saught AI widget in a clean environment to help debug styling issues.

Test Environment

The widget should appear in the bottom-right corner of this page.

Widget version: v0.js

Check the browser console for any errors.

Expected Behavior

  • Widget should use Inter font (similar to Geist)
  • Shadows should be visible on the widget button and dropdown
  • All Tailwind utility classes should work properly
  • The widget should be isolated in a Shadow DOM

Debug Info

Open DevTools and inspect the #saught-widget-root element.

Check the Shadow DOM for proper CSS injection.
