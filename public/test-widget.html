Saught AI Widget Test Page

Welcome to the Saught Widget Demo

This page demonstrates the Saught AI widget. Look for the "Ask AI" button in the bottom-right corner of this page.

The widget allows you to ask questions about the content on this page and get answers from your preferred AI service.

How it works

The widget is loaded via a simple script tag:

<script src="v1.js"></script>

Once loaded, it automatically appears on your page with a floating button that users can click to ask questions.

Features

  • Choose from multiple AI services (ChatGPT, Claude, Perplexity, Copilot, Grok)
  • Quote text from the page by selecting it and clicking the quote button
  • The widget remembers your preferred AI service
  • Clean, modern UI that works on any website
  • Isolated in a Shadow DOM to prevent style conflicts

Configuration Options

You can customize the widget by adding data attributes to the script tag:

<script 
    src="v1.js"
    data-default-ai="claude"
    data-placeholder="Ask me anything..."
    data-agent-prompt="Custom prompt template"
></script>
